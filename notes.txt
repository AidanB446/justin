
INSTALLATION -
	- project written and made in linux, install and build may be different on windows and macos
	- ensure system has python tool installed, and is available in the terminal.
	- run the ./build.sh script
	- then source the venv, 
		source ./venv/bin/activate

	- to run, run the ./run.sh script

Market Order - 
	Order executes immediately, not guarenteed you will get the price you 
	believe. 

Limit Order - 
	sell or buy at exactly this price
	returns a status, does not execute immediately


- AI DESCRIPTION FOR TRADING MINDSET - 
		How these fit together (mental model)

		When a trader looks at a stock, their brain does this:

		Is the market open?

		What’s the current bid/ask?

		What was the last trade price?

		What’s the short-term trend (1m/5m candles)?

		Is volume confirming the move?

		That’s the decision loop.


- Limit Orders - 
	sell at or buy at this price point, 	

- Market Orders
	sell or buy instantly at whatever market price is.
